Movie API

Description

The myFlix web application provides users with access to a vast library of movie information, including details about different movies, directors, and genres.

Users can register, create a profile, and manage their favorite movies list. Additionally, users can explore actor information, view movie ratings, and create a "To Watch" list for future movie selections.

With its user-friendly interface and comprehensive database, myFlix offers an immersive experience for movie enthusiasts to discover, explore, and engage with their favorite films.

Users

Method URL Description Parameters
POST /users Creates a new user. None
Body of the
Request		 
{
    "name": "Jorge",
    "favoriteMovies": []
}
Response 
{
    "name": "Jorge",
    "favoriteMovies": [],
    "id": "09896e6e-e7a1-4202-95bc-6fcf78957231"
}
PUT /users/:id Updates an existing user.
:id
The id of the user
Body of the
Request		 
{
    "name": "Jorge (update)"
}
Response 
{
    "id": "09896e6e-e7a1-4202-95bc-6fcf78957231",
    "name": "Jorge (update)",
    "favoriteMovies": []
}
POST /users/:id/movies/:title Adds a movie to a user's favorite movies list.
:id
The id of the user
:title
The title of the movie
Response 
The Dark Knight has been added to user 09896e6e-e7a1-4202-95bc-6fcf78957231's array
DELETE /users/:id/movies/:title Removes a movie from a user's favorite movies list.
:id
The id of the user
:title
The title of the movie
Response 
The Dark Knight has been removed from user 09896e6e-e7a1-4202-95bc-6fcf78957231's array
DELETE /users/:id Deletes a user from the system.
:id
The id of the user
Response 
User 09896e6e-e7a1-4202-95bc-6fcf78957231

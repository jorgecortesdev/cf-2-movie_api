Movie API

Description

The myFlix web application provides users with access to a vast library of movie information, including details about different movies, directors, and genres.

Users can register, create a profile, and manage their favorite movies list. Additionally, users can explore actor information, view movie ratings, and create a "To Watch" list for future movie selections.

With its user-friendly interface and comprehensive database, myFlix offers an immersive experience for movie enthusiasts to discover, explore, and engage with their favorite films.
